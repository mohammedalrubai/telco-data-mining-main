# 📊 Telco Customer Churn — Data Mining Project

## 🌍 English

### 🧭 Overview

This project explores a Telco customer dataset and builds machine learning models to predict customer churn (whether a customer will leave the service).

### 🎯 Objective

* Analyze customer behavior
* Identify factors influencing churn
* Build predictive models

### 📂 Dataset

* File: `telco_customer_churn.csv`
* Includes:

  * Customer info
  * Services & subscriptions
  * Billing details
  * Churn label

### 🔎 Workflow

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training:

  * Logistic Regression
  * Random Forest
  * XGBoost
* Model Evaluation

### 🚀 How to Run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook data_mining_telco_FINAL.ipynb
```

### 💡 Notes

* Use stratified split for better evaluation
* Try feature importance or SHAP for explainability
